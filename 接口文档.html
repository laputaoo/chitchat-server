接口文档


1.注册
地址: http://localhost:3000/register

请求方式: POST

参数:

字段 类型 说明 必需
nick_name string 用户名 是
phone string 手机号 是
password string 密码 是

返回值:

字段 类型 说明 必需
code number 0:成功 1:失败 是
msg string 返回信息 是
2.密码登录
地址: http://localhost:3000/login

请求方式: POST

参数:

字段 类型 说明 必需
phone string 手机号 是
password string 密码 是

返回值:

字段 类型 说明 必需
code number 0:成功 1:失败 是
nick_name object 返回用户昵称 是
imgUrl object 用户头像 是

参数:

字段 类型 说明 必需
phone string 手机号 是

返回值:

字段 类型 说明 必需
code number 0:成功 1:失败 是
msg string 返回信息 是
4.短信登录
地址: http://localhost:3000/msgLogin

请求方式: POST

参数:

字段 类型 说明 必需
phone string 手机号 是
verifyCode string 短信验证码 是

返回值:

字段 类型 说明 必需
code number 0:成功 1:失败 是
nick_name object 返回用户昵称 是
imgUrl object 用户头像 是
5.搜索用户判断是否好友
地址: http://localhost:3000/search

请求方式: POST

参数:

字段 类型 说明 必需
phone string 手机号 是
userID string 用户ID 是

返回值:

字段 类型 说明 必需
code number 0:成功 1：失败 是
state number 0:不是好友 1:是好友 2：是自己 是
nick_name object 返回用户昵称 是
imgUrl object 用户头像 是
6.申请好友

请求方式: POST

参数:

字段 类型 说明 必需
userID string 用户ID 是
friendsID string 朋友ID 是
application string 申请词 是
markName string 备注 是

返回值:

字段 类型 说明 必需
code number 0:成功 是
